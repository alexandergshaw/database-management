# Week 3 ERD Design

## Learning Objective
Create an entity relationship schema object for the enterprise domain.

## GUI-Only Workflow
1. Open this repository in **GitHub Codespaces**.
2. In the editor, update only `/assignments/assignment2/solution.ts`.
3. Use the **Testing panel** to run `/assignments/assignment2/test.ts`.
4. Use the **Source Control panel** to stage and commit.
5. Push from the Source Control panel and open a GitHub pull request.
6. Verify the Vercel preview before merging.

## Notes
- Keep changes inside this assignment folder.
- The dashboard unlocks when `result.isComplete` is set to `true` and your export is valid.
